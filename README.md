# VMS — интерактивный прототип раскладки камер

Прототип интерфейса видеомониторинга с 4 ячейками и таймлайном записей.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

## Функциональность

- **4 ячейки** — каждая показывает название камеры и статус (Live / Archive с временем)
- **Выбор ячейки** — клик по ячейке выделяет её зелёной рамкой; выбор другой снимает выделение с предыдущей
- **Таймлайн** — блоки записей за 24 часа, клик переключает воспроизведение
- **Синхронизация ячеек** — тогл над таймлайном:
  - **Включён**: клик по записи переключает все 4 ячейки на это время
  - **Выключен**: выбираете ячейку → кликаете по записи → управляете только ей; при выборе другой ячейки предыдущая переходит в Live
